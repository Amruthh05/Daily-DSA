# Day 05

## Problem: Add Binary

Platform: LeetCode  
Problem Number: 67  
Difficulty: Easy  
Topic: String, Bit Manipulation

### Problem

We are given two binary strings `a` and `b`.

Our task is to return the **sum of these two binary numbers** as a binary string.

Binary numbers contain only `0` and `1`.

### Example 1

Input  
a = "11"  
b = "1"

Output  
"100"

Explanation  
11 (binary) = 3  
1 (binary) = 1  

3 + 1 = 4  
Binary of 4 is **100**

---

### Example 2

Input  
a = "1010"  
b = "1011"
