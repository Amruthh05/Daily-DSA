# Day 10

## Problem: The Two Sneaky Numbers of Digitville

Platform: LeetCode  
Problem Number: 3289  
Difficulty: Easy  
Topic: Array, Hash Set

### Problem

In Digitville, there is a list of numbers `nums` containing integers from `0` to `n - 1`.

Normally, each number should appear exactly **once**.

However, **two numbers appear twice**, making the list longer than expected.

Your task is to **find those two sneaky numbers**.

Return the two numbers in any order.

---

### Example 1

Input  
nums = [0,1,1,0]

Output  
[0,1]

Explanation  
Both 0 and 1 appear twice.

---

### Example 2

Input  
nums = [0,3,2,1,3,2]

Output  
[2,3]

Explanation  
Both 2 and 3 appear twice.
