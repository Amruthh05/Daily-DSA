
# Day 13

## Problem: First Letter to Appear Twice

Platform: LeetCode  
Problem Number: 2351  
Difficulty: Easy  
Topics: String, Hashing

---

### Problem Statement

Given a string `s` consisting of lowercase English letters, return the **first letter to appear twice**.

A letter appears twice before another letter if its **second occurrence comes first**.

---

### Example 1

Input  
s = "abccbaacz"

Output  
"c"

Explanation  
'c' is the first letter whose second occurrence appears earliest.

---

### Example 2

Input  
s = "abcdd"

Output  
"d"

Explanation  
'd' is the only character that appears twice.
