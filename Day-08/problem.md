# Day 08

## Problem: Move Zeroes

Platform: LeetCode  
Problem Number: 283  
Difficulty: Easy  
Topic: Arrays

### Problem

We are given an integer array `nums`.

Our task is to **move all the zeroes to the end of the array** while maintaining the relative order of the non-zero elements.

Important:

- We must do this **in-place**.
- We should **not create another copy of the array**.

---

### Example 1

Input  
nums = [0,1,0,3,12]

Output  
[1,3,12,0,0]

Explanation  
All non-zero elements are moved to the front and zeros are placed at the end.

---

### Example 2

Input  
nums = [0]

Output  
[0]
