
# Day 01

## Problem: Two Sum

Platform: LeetCode  
Difficulty: Easy  
Topic: Array, Hash Map

### Problem

We are given an array of integers called `nums` and a number called `target`.

We need to return the **indices of two numbers** in the array such that their sum is equal to the target.

We can assume that:
- There is exactly one correct answer.
- We cannot use the same element twice.
- The order of the output does not matter.

### Example

Input:
nums = [2,7,11,15]  
target = 9  

Output:
[0,1]

Explanation:  
nums[0] + nums[1] = 2 + 7 = 9
