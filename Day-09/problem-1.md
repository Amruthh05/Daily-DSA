# Day 09

## Problem: Plus One

Platform: LeetCode  
Problem Number: 66  
Difficulty: Easy  
Topic: Arrays, Math

### Problem

We are given an integer array `digits`.

The array represents a large number where:

- Each element is a digit.
- The digits are ordered from **most significant to least significant**.

Our task is to **add one to the number** and return the resulting array of digits.

The number will not contain any leading zeros.

---

### Example 1

Input  
digits = [1,2,3]

Output  
[1,2,4]

Explanation  
The array represents the number **123**.  
123 + 1 = **124**

---

### Example 2

Input  
digits = [4,3,2,1]

Output  
[4,3,2,2]

Explanation  
4321 + 1 = **4322**

---

### Example 3

Input  
digits = [9]

Output  
[1,0]

Explanation  
9 + 1 = **10**
